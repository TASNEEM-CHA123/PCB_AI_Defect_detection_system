# Skills Guide: Industrial AI Specialist (PCB Domain)

To successfully implement this project, the following skills are required:

## 1. Computer Vision & Architectures
- **NMS-Free YOLO (Ultralytics 2026):** Understanding the elimination of Non-Maximum Suppression for CPU efficiency.
- **SAM 2 Prompt Engineering:** Mastering how to guide a foundation model using bounding box coordinates.
- **Small Target Optimization:** Experience with SAHI (Slicing Aided Hyper Inference) for microscopic defect detection.

## 2. Advanced Python & Frameworks
- **Streamlit:** For rapid deployment of professional-grade dashboard interfaces.
- **PyTorch/OpenVINO:** For model deployment and cross-platform inference optimization.
- **OpenCV:** Core image manipulation, color space conversion, and alpha blending.

## 3. Industrial Data Management
- **Dataset Benchmarking:** Proficiency in working with PKU-Market-PCB and COCO-style JSON annotations.
- **Audit Compliance:** Understanding how to translate pixel masks into real-world area measurements (mm²).
